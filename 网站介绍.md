# 《电气朋友圈》网站介绍

## 1. 项目定位
《电气朋友圈》是一款面向《电气工程学概论》课程展示的互动式前端网页项目。  
我们用“社交媒体信息流”的方式重构电气工程发展史，让历史人物、关键技术与现代学生在同一界面中展开跨时代对话。

## 2. 核心创意
传统电气工程史通常按时间线和章节讲述，信息完整但参与感较弱。  
本项目将历史人物设计成“账号用户”，把关键发现和发明设计成“动态发布”，通过评论区互动和话题标签机制，让知识点之间形成可视化“技术接力”。

在这个叙事里，富兰克林、伏打、奥斯特、安培、法拉第、麦克斯韦、赫兹、爱迪生、特斯拉等角色不再只是教材中的名字，而是具有观点、语气和工程语境的“发帖者”。

## 3. 主要功能模块
1. 首页信息流  
展示历史动态卡片，支持搜索、话题筛选、互动状态显示与右侧榜单。

2. 动态详情页  
支持完整长文阅读、目录锚点、阅读进度条、回到顶部、评论区互动。

3. 时间轴页面  
按历史顺序展示关键节点，并提供“讲解路径”联动，可跳转并高亮对应动态、人物和发明。

4. 人物页与人物主页  
展示关键角色卡片、人物简介、代表贡献及其关联动态。

5. 发明图鉴页  
从工程应用视角组织关键发明，突出“理论—装置—系统—产业”的演化逻辑。

6. 登录/注册  
使用 localStorage 模拟用户注册、登录与互动持久化，无后端也可完整演示。

## 4. 技术实现
- 前端框架：React + TypeScript + Vite
- 路由系统：React Router
- 样式方案：Tailwind CSS + 自定义全局样式
- 数据层：TypeScript mock 数据 + selector 查询封装
- 状态持久化：localStorage（登录态、互动、评论）

项目结构采用数据层、组件层、页面层分离，便于后续平滑接入 Node.js + Express + PostgreSQL。

## 5. 设计与体验特点
- 保持课堂投影可读性：高对比度、清晰字体、信息密度可控
- 移动端专属优化：底部导航、搜索区重排、按钮交互优化
- 视觉语言统一：科技感蓝紫主题、卡片层次、轻量动效
- 头像系统升级：支持人物真实头像与圆形裁切显示

## 6. 教学价值
《电气朋友圈》不只是“把历史搬到网页上”，而是通过可互动叙事帮助学习者自然理解以下主线：
- 电生磁
- 磁生电
- 理论统一
- 工程落地
- 信息传输
- 系统智能化

这使抽象原理更易进入具体情境，也让期中展示更具故事性、连贯性与传播效果。

## 7. 后续可扩展方向
- 接入真实后端与数据库，支持多用户在线互动
- 增加课堂演示模式（一键大字号、讲解脚本、自动高亮路径）
- 引入数据可视化（技术关系图谱、发明影响网络）
- 增加单元测试与端到端测试，提升展示稳定性
